# 🚢 ReadyPI Deployment Guide

ReadyPI is designed to run on **Google Cloud Platform (GCP)** using **Cloud Run** and **Cloud SQL**.

## 🏗️ Automated Infrastructure Setup
We provide a comprehensive script to provision all necessary resources.

### Prerequisites
1. Install [gcloud CLI](https://cloud.google.com/sdk/docs/install).
2. Authenticate: `gcloud auth login`.
3. Set your project: `gcloud config set project [PROJECT_ID]`.

### Run Provisioning
```bash
chmod +x gcp-init.sh
./gcp-init.sh
```
This script will:
- Enable all required APIs (Cloud Run, SQL, Secret Manager).
- Create a PostgreSQL 15 instance.
- Setup Secret Manager placeholders.
- Configure IAM roles & service accounts.

## 🔑 Secret Management
ReadyPI uses **GCP Secret Manager** in production. Ensure you set the following secrets before deploying:

```bash
# Example: Setting Groq API Key
echo -n "your-key" | gcloud secrets versions add GROQ_API_KEY --data-file=-
```

**Required Secrets:**
- `DB_PASSWORD` (Auto-generated by gcp-init.sh)
- `JWT_SECRET` (Auto-generated by gcp-init.sh)
- `GROQ_API_KEY`
- `GOOGLE_API_KEY`
- `SSLCOMMERZ_STORE_ID`
- `SSLCOMMERZ_STORE_PASSWORD`

## 🚀 CI/CD Pipeline
Deployment is automated via **Google Cloud Build**.

1. Connect your GitHub repository to Google Cloud Build.
2. Create a trigger based on the `cloudbuild.yaml` file.
3. Push to `main` branch to trigger a production build and deploy.

### Manual Deployment
If you prefer to deploy manually:
```bash
# Build & Push Docker Image
docker build -t gcr.io/[PROJECT_ID]/readypi-api ./api
docker push gcr.io/[PROJECT_ID]/readypi-api

# Deploy to Cloud Run
gcloud run deploy readypi-api \
  --image gcr.io/[PROJECT_ID]/readypi-api \
  --platform managed \
  --add-cloudsql-instances [INSTANCE_CONNECTION_NAME] \
  --set-secrets=DB_PASSWORD=DB_PASSWORD:latest,JWT_SECRET=JWT_SECRET:latest \
  --region asia-southeast1
```

## 🌐 Dashboard (Next.js)
The dashboard is also deployed to Cloud Run.

**Environment Variables:**
- `NEXT_PUBLIC_API_URL`: URL of your deployed Express API.
- `NEXT_PUBLIC_FIREBASE_CONFIG`: Your Firebase project configuration.
